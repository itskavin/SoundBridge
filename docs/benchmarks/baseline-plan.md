# Baseline Benchmark Plan

## Target Metrics

- Connection success rate
- p50/p95 end-to-end latency
- Jitter and dropout count
- CPU and battery impact

## Matrix

- Windows, macOS, Linux, Android, iOS
- Wi-Fi 2.4 GHz, 5 GHz, 6 GHz, USB tether
- Voice and music profiles

## Method

- 30 repeated runs per scenario
- Store raw data and summary table in version control
- Publish comparative trend chart per release
