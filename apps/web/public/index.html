SoundBridge Web P2P

Web-first, no managed media relay. Browser peers connect directly over WebRTC using manual offer/answer exchange.

Quick Start

  1. Open this page in two devices or browsers.
  2. On device A, use Broadcaster and copy the offer.
  3. On device B, paste offer, create answer, then copy answer.
  4. Back on device A, paste answer and apply it.

Recommended for local testing: same Wi-Fi network, HTTPS in production.
